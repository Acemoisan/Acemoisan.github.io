What is 2140?
2140 is a Zombie Survival Game that I have created as a solo development team for the past 12 months. I have created all assets and programming myself. Including all Models. 2D Images. UI. Rigging. VFX and more! I have implemented Play to Earn into my survival game. And released it on various Play To Earn Platforms! I have NFTs available to be purchased and traded. Even used in game! All NFTs have various attributes in game. They can be chosen at the start of the game to make every experience different! I have begun to create a community for this game. Using Discord!
You can find all of my additional info for the game using the URL above!
